วิธีการขุดเหรียญ DYNAX

  1. ติดตั้ง Termux บน Android
  2. Clone repository: git clone https://github.com/yourusername/dynax.git
  3. รัน Node: python3 dynax_node_v20.py
  4. ส่งธุรกรรมไปที่ endpoint /tx
  5. ขุด Block: curl http://127.0.0.1:6002/mine/YOUR_ADDRESS
  6. รับรางวัล 50 DYX ต่อ Block

รายละเอียดทางเทคนิค

อัลกอริทึม: SHA3-256 Proof of Work

ความยาก: 4 zeros (0000)

รางวัลต่อ Block: 50 DYX

เวลาต่อ Block: ~1-2 minutes

ลายเซ็น: ECDSA + SHA3-256

รูปแบบ Address: DX + SHA3-256(pubkey)[:40]

Q: ขุดคนเดียวหรือรวมกลุ่ม?
A: ขุดคนเดียว (Solo Mining)

Q: ใช้ไฟฟ้าเท่าไหร่?

บทนำ

DYNAX เป็นบล็อกเชนแบบกระจายอำนาจ Layer 1 ที่ออกแบบมาเพื่อทนทานต่อการโจมตีจากคอมพิวเตอร์ควอนตัม

คุณสมบัติหลัก

  • ไร้ตัวกลาง: ไม่มีเซิร์ฟเวอร์กลางควบคุมเครือข่าย
  • ปลอดภัย: ใช้ ECDSA + SHA3-256 ในการตรวจสอบลายเซ็น
  • ทนทานต่อควอนตัม: ออกแบบมาเพื่อต้านทานการโจมตีจากควอนตัมคอมพิวเตอร์
  • ไม่มีการระดมทุน ICO: กระจายเหรียญผ่านการขุดเท่านั้น

โครงสร้างทางเทคนิค

  • รูปแบบ Address: DX + SHA3-256(public_key)[:40]
  • ธุรกรรม: ตรวจสอบลายเซ็น ECDSA
  • ฉันทามติ: Proof of Work (SHA3-256)
  • เวลา Block: ปรับความยากแบบไดนามิก

โทเคนโนมิกส์

  • อุปทานรวม: ไม่จำกัด (ผ่านทางการขุด)
  • รางวัลต่อ Block: 50 DYX
  • ค่าธรรมเนียม: ผู้ใช้กำหนดเอง
  • Halving: ไม่มี (รางวัลคงที่)

แผนการพัฒนา

  • Phase 1: การใช้งานบล็อกเชนพื้นฐาน ✅
  • Phase 2: เว็บ Wallet และ UI ✅
  • Phase 3: การใช้งานระบบ P2P Network 🔄
  • Phase 4: พัฒนาแอปมือถือ 📋
  • Phase 5: รองรับ Smart Contracts 📋
